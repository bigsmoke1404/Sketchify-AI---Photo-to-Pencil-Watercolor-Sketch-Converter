What is Image Processing?

Image processing is a method to perform some operations on an image, in order to get an enhanced image or to extract some useful information from it. It is a type of signal processing in which input is an image and output may be image or characteristics/features associated with that image.

What is OpenCV?

OpenCV (Open Source Computer Vision Library) is an open source computer vision and machine learning software library. OpenCV was built to provide a common infrastructure for computer vision applications and to accelerate the use of machine perception in the commercial products. It contains more than 2500 optimized algorithms!

How Pencil Sketch Works

The pencil sketch effect is achieved using cv2.pencilSketch(). It works by exploiting the spatial and range filters to preserve edges while smoothing out the internal regions, simulating the strokes of a pencil. The two main parameters are:

  • sigma_s: Controls the spatial size of the neighborhood.
  • sigma_r: Controls how dissimilar colors within the neighborhood will be averaged.

How Watercolor Works

The watercolor effect uses a combination of cv2.edgePreservingFilter() and cv2.stylization(). First, it smooths the image while strictly maintaining strong edges. Then, the stylization filter creates the characteristic "color bleed" and saturated look typical of watercolor paintings.
